	Awesome Product Slider for query post from any post type and display on grid or Slider.

== Description ==

Query post from your custom post types and create awesome Slider.

Use any where by short-codes and easy customize via short-code generator page, you can display or hide grid meta items like post authors, post date, post categories, post tags and etc.

== Installation ==

1. Install as regular WordPress plugin.<br />
2. Go your plugin setting via WordPress Dashboard and find "<strong>Product Slider</strong>" activate it.<br />

After activate plugin you will see "Product Slider" menu at left side on WordPress dashboard click "New Product Slider" and use the options field "Product Slider"<br />

<br />
<strong>How to use on page or post</strong><br />
When Product Slider options setup done please publish Product Slider as like post or page<br />

and then copy shortcode from top of <strong>Product Slider Options</strong> `[product_slider  id="1234" ]`<br />

then paste this shortcode anywhere in your page to display grid<br />
